Who we are


Web Consumption is a B2B (business-to-business) web operator, which offers web services in the field of resource management (workforce & field service management, property management, nightly accommodation reservations, hourly job reservations), optionally including physical labor dispatching (controlled via the web services), to European clients.


Our Values


Web Consumption differs from common web platforms which do only brokerage services as an intermediary agent for a commission, Web Consumption runs a model of buying the labor from a supplier and then re-selling it as the main provider to the customer, with a profit margin.

Web Consumption operates a web server with software code installed on it. But it does not own the software code (& domain names & trademarks & marketing material), but leases a license from Sharing Inventions Ltd., which enables Web Consumption Ltd. to use it for the European market.
